# IQ - Fun IQ Test

A simple, fun, and interactive IQ test built with HTML, CSS, and JavaScript.

## Overview

The Fun IQ Test application provides a quick way to test your cognitive abilities across different categories, including Math, Memory, and Logic. The test is designed to be interactive and provides an estimated IQ score based on your performance and speed.

## Features

- **Math Questions:** Randomized addition and multiplication problems.
- **Memory Challenges:** Test your ability to recall numeric sequences shown briefly.
- **Logic Puzzles:** Simple true/false logic questions (e.g., "Is zero an even number?").
- **Speed Scoring:** Points are awarded for both correctness and speed.
- **Estimated Results:** Get an estimated IQ score based on a standard distribution.

## How to Run

Since this is a single-file web application, you can run it directly in your web browser:

1.  Navigate to the project directory.
2.  Open `iq.html` in any modern web browser.
3.  Click "Start Test" to begin.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS with gradients and responsive design)
- JavaScript (Vanilla JS for application logic and scoring)
